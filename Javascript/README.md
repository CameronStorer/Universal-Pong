# Web Pong
A simple implementation of the classic arcade game Pong written in JavaScript + HTML.

## Requirements
Have a web browser.

## Instructions
To run this application, open the index.html file with your preferred browser. Additionally, you can play it at [pong.cameronstorer.com](https://pong.cameronstorer.com).

## Notice
The game tab will open and display a live, interactive session of the classic arcade game Pong.
