# Python Pong
A simple implementation of the classic arcade game Pong written in Python.

## Requirements
You must have a Python installation on your system.

This project uses the Pygame framework. After installing Python, install the dependency by running:
```pip install pygame```

## Instructions
To run this python application, open a terminal in the project directory and run:
```python main.py```

## Notice
The game window will open and display a live, interactive session of the classic arcade game Pong.
